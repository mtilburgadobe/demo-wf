The Skills

What each skill audits

🗺️

Sitemap Checker

URL availability audit

Fetches the live site's sitemap and checks every URL against the EDS domain via HTTP HEAD requests. Identifies pages that are live, missing, or redirected.

  • ✓ 200 OK — live and accessible
  • ✓ 404 Not Found — not yet published
  • ✓ 301/302 Redirects — rules to review
  • ✓ Timeouts and connectivity errors
10 concurrent · ~1 min / 100 URLs
📝

Content Validator

Content quality & parity

Three modes: fast parity comparison (HTTP fetch), deep comparison (Playwright + lazy loading), and site audit (EDS-only quality gate).

  • ✓ Placeholder / lorem ipsum text detection
  • ✓ Missing image alt text & broken image URLs
  • ✓ Generic anchor text ("click here", "read more")
  • ✓ Stub pages, missing H1, nav/footer health
  • ✓ Content similarity score vs production (Jaccard %)
Fast Deep Site Audit
🔍

SEO Validator

SEO health & performance

Four modes from basic metadata to full Lighthouse + real-user field data. The deep mode covers 68 of 80 items from the SEO Validation Checklist using a real Chromium browser.

  • ✓ Title, description, OG/Twitter tags (length + presence)
  • ✓ JSON-LD structured data, canonical, heading hierarchy
  • ✓ Core Web Vitals: CLS, LCP, FCP, TTFB (synthetic)
  • ✓ PageSpeed Insights + CrUX field data incl. INP
  • ✓ Mobile responsiveness, favicon, image accessibility
Fast Deep CWV / PageSpeed Compare
🖼️

Visual Compare

Screenshot regression testing

Full-page Playwright screenshots at desktop (1440px), tablet (768px), and mobile (390px), diffed pixel-by-pixel with pixelmatch. Changed pixels are highlighted in red.

  • ✓ Desktop, tablet, and mobile viewports
  • ✓ Full-page scroll to capture lazy-loaded content
  • ✓ Diff % per page per viewport
  • ✓ Side-by-side Prod | EDS | Diff HTML report
  • ✓ Height delta flagging (missing sections)
3 viewports · ~5 min / 10 pages

Deliverables

Every audit produces shareable reports

All skills output both a CSV (for data analysis) and a self-contained HTML file (for stakeholder sharing). HTML reports include interactive tables, colour-coded pass/fail status, and per-issue category badges — no tool installation required to view them.

CSV Reports

Per-page rows with all metric columns. Import into Excel, Google Sheets, or any BI tool for custom filtering and prioritisation.

HTML Reports

Self-contained browser files with stat cards, sorted tables, colour-coded badges, and (for visual compare) embedded screenshot panels. Shareable by email or Slack.
